    editor/Workspace

    Рабочее пространство. Компонент предназначен масштабирования и скролинга документа.

    Props

    Name Type Default value Required? Description
    rect Object - Yes Размеры рабочей области.

    Data

    Name Type Default value Description
    mouseOver Boolean - true если курсор находится над рабочей областью.
    showGrid Boolean - true если необходимо отображать сетку.
    offset Point - Отступ от курсора до документа. null если drag and drop не активен.
    zoom Number - Уровень зума. Не работает.
    aspectRatio Number - Соотношение ширины к высоте документа.
    position Point - Текущая позиция документа.

    Computed

    Name Type Description
    documentStyle Object Стиль для документа.
    grid Object Сетка. Просто сетка.
    gridStyle Object Стиль для сетки.
